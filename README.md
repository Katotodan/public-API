This is a public API that generate an email address, current time and github link for this repository.
<br/>
The app is build with node.js and express.js.
In oder to run this app locally, clone this repo and then run <strong>npm install</strong> to install all
dependecies. Then start the app by running the command <strong>node index.js</strong>.
You must have node.js and npm install in your PC first.
